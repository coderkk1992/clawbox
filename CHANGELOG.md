# Changelog

All notable changes to ClawBox will be documented in this file.

## [0.1.0] - 2025-03-23

### Added

- Initial release
- One-click setup wizard with resource allocation (RAM, CPU)
- VM isolation using Lima (Ubuntu 24.04)
- Built-in chat interface with OpenClaw agent
- Image paste from clipboard (Cmd+V)
- Drag and drop file uploads
- Multi-channel support (Telegram, Discord, Slack, and more)
- System tray integration
- Agent personality customization
- Automatic Lima download (no Homebrew required)

### Supported Platforms

- macOS 13+ (Ventura) on Apple Silicon
- macOS 13+ (Ventura) on Intel
